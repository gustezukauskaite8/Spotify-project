# Analysis of Spotify Data 2023

Explanation of the file layout:

- Files containing “ANALYSIS” include code for Part 1 of the assignment, focusing on basic data exploration and initial insights.
- Files containing “CHOSEN” include code for Part 3. Each group member selected an album and a feature to investigate, along with additional analysis of explicit tracks, data relationships, and collaborations.
- Files containing “TAB” include code used to build the final dashboard. The dashboard covers album, artist and genre analysis.
